# JavaScript Form Validation & Character Counter

## Overview

This project consists of two mini web applications built using HTML, CSS, and JavaScript.

### Project 1: Registration Form Validation

A registration form that validates user input in real time. The form includes the following fields:

* Name
* Email
* Password

### Features

* Validates that the Name field is not empty.
* Validates that the Email field follows a proper email format.
* Validates that the Password contains at least 6 characters.
* Displays error messages for invalid inputs.
* Enables the Submit button only when all fields are valid.
* Provides real-time feedback while the user types.

---

## Project 2: Character Counter

A text area that tracks the number of characters entered by the user in real time.

### Features

* Displays a maximum character limit of 200 characters.
* Updates the character count as the user types.
* Shows the number of characters entered.
* Displays the remaining characters available.
* Prevents users from exceeding the character limit.
* Displays a warning message when the limit is reached.
